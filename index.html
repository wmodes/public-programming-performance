A Public Programming Performance

About

Public Programming Performance is an in-class, live collaborative programming experiment where a few volunteer students will code at the front of the class. Using VSCode Live Edit and a shared GitHub repo, these volunteers will write code live while collaborating with classmates. Volunteers can seek assistance from their peers during the session and have the option to tag out, allowing another student to take over their position. This setup encourages active participation and collective problem-solving, as the entire class engages directly in the programming process, contributing ideas and solutions in real-time.

Boaty McLongBoat" ⇒ "Island Archipelago

We're starting with a simple "Infinite Worlds" sketch derived from Experiment 04. Boaty McLongBoat is an infinite boat in which users can rearrange the deck chairs.

We'll be removing McLongBoat and creating an island archipelago. Features:

  1. Islands with colored geography
  2. Users can navigate this infinite world with arrow keys.
  3. Users can place boats in the water.
  4. The world seed affects the island placement

Possible stretch goals:

  1. Islands have varrying geography?
  2. Boats move around on the water?
  3. Adjacent boats make longer, wider, bigger boats?
  4. People can be placed on the islands?
  5. People move around on the islands?
  6. Splash animations when boats are placed or removed?
  7. What happens when you click on an island?
